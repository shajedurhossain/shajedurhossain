## 📌 Featured Projects

### 🌾 [Optimisation of Best Progeny Value in Recurrent Genomic Selection](https://github.com/shajedurhossain/genomic-selection-bpv)
> MSc Thesis | JLU Gießen | 2020 – Present

Investigating weighting factor strategies to optimise Best Progeny Value (BPV) across recurrent genomic selection cycles. Implements and benchmarks multiple genomic prediction models in R to evaluate selection accuracy and breeding cycle efficiency.

- Genomic prediction modelling using R and Bioconductor tools
- Comparison of BPV weighting schemes using simulation and real breeding data
- Practical application to crop improvement programmes
- **Tools:** R, rrBLUP / BGLR, ggplot2, HPC

---

### 🌿 [Transcriptomic Response to Cold Stress in Sorghum](https://github.com/shajedurhossain/sorghum-cold-stress-rnaseq)
> Previous Research Project | JLU Gießen

End-to-end bulk RNA-seq analysis identifying cold-stress response genes in *Sorghum bicolor*, from raw reads through to pathway enrichment.

- Full pipeline: raw FASTQ → QC → alignment → DESeq2 differential expression → GO/KEGG enrichment
- Identified key cold-responsive genes and enriched biological pathways
- **Tools:** R, DESeq2, STAR, clusterProfiler, ggplot2

---

### 🧬 [Bioinformatics Utility Scripts](https://github.com/shajedurhossain/bioinfo-utils)
> A collection of practical scripts for common genomics tasks

Python and Bash utilities for FASTQ QC parsing, VCF filtering, FASTA manipulation, and HPC job submission automation.
